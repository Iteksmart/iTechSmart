## üß™ Testing Guide

### 1. Local Testing

#### Test Individual Products
```bash
# Navigate to product directory
cd itechsmart-ninja

# Install dependencies
npm install  # or pip install -r requirements.txt

# Run tests
npm test  # or pytest

# Build locally
npm run build  # or python setup.py build
```

#### Test Docker Builds
```bash
# Build Docker image
docker build -t itechsmart-ninja:1.5.0 -f itechsmart-ninja/Dockerfile .

# Run container
docker run -d -p 8080:8080 itechsmart-ninja:1.5.0

# Test endpoints
curl http://localhost:8080/health
curl http://localhost:8080/api/v1/status
```

### 2. Integration Testing

#### Test Product Integrations
```bash
# Start all services with docker-compose
docker-compose up -d

# Run integration tests
pytest tests/integration/

# Check service health
./scripts/health_check.sh
```

#### Test API Endpoints
```bash
# Test Ninja API
curl -X GET http://localhost:8080/api/v1/workflows

# Test Enterprise API
curl -X GET http://localhost:8081/api/v1/compliance

# Test Analytics API
curl -X GET http://localhost:8082/api/v1/insights
```

### 4. Performance Testing

#### Load Testing
```bash
# Install Apache Bench
sudo apt-get install apache2-utils

# Test API performance
ab -n 1000 -c 10 http://localhost:8080/api/v1/health

# Test with authentication
ab -n 1000 -c 10 -H "Authorization: Bearer TOKEN" http://localhost:8080/api/v1/workflows
```

#### Stress Testing
```bash
# Install k6
curl https://github.com/grafana/k6/releases/download/v0.45.0/k6-v0.45.0-linux-amd64.tar.gz -L | tar xvz

# Run stress test
k6 run tests/stress/api_stress_test.js
```

### 5. Security Testing

#### Vulnerability Scanning
```bash
# Scan Docker images
docker scan itechsmart-ninja:1.5.0

# Scan dependencies
npm audit  # or pip-audit
safety check  # Python dependencies
```

#### Security Compliance
```bash
# Run security tests
pytest tests/security/

# Check for secrets
trufflehog --regex --entropy=False .

# SAST scanning
bandit -r . -f json -o bandit-report.json
```
