```bash

  -H "Content-Type: application/json" \

    "context": {}
  }'
```

### Create App
```bash
curl -X POST "http://localhost:8320/api/apps?user_id=1" \
  -H "Content-Type: application/json" \
  -d '{
    "name": "My CRM",
    "description": "Customer relationship management"
  }'
```

```bash
  -H "Content-Type: application/json" \
```
