## ðŸ“¦ Installation & Upgrade

### New Installation

```bash
# Pull latest images
docker-compose pull

# Start services
docker-compose up -d

# Run migrations
docker-compose exec compliance alembic upgrade head
docker-compose exec enterprise alembic upgrade head
docker-compose exec workflow alembic upgrade head
docker-compose exec observatory alembic upgrade head
docker-compose exec analytics alembic upgrade head

# Verify deployment
curl http://localhost:8019/health
```

### Upgrade from v1.0

```bash
# Backup database
docker-compose exec postgres pg_dump -U itechsmart itechsmart > backup.sql

# Stop services
docker-compose down

# Update docker-compose.yml with v2.0 images

# Start services
docker-compose up -d

# Run migrations
docker-compose exec compliance alembic upgrade head
docker-compose exec enterprise alembic upgrade head
docker-compose exec workflow alembic upgrade head
docker-compose exec observatory alembic upgrade head
docker-compose exec analytics alembic upgrade head

# Verify upgrade
docker-compose ps
```

### Rollback Procedure

```bash
# Stop services
docker-compose down

# Restore v1.0 configuration
docker-compose -f docker-compose.v1.yml up -d

# Restore database
docker-compose exec postgres psql -U itechsmart itechsmart < backup.sql

# Verify rollback
docker-compose ps
```
