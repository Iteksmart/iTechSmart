## ğŸš€ Quick Start

### Prerequisites
- Python 3.11+
- Node.js 20.x+
- VS Code (latest)
- API keys (OpenAI, Anthropic, Google minimum)

### Installation (5 Minutes)

```bash
# 1. Install backend dependencies
cd backend
python -m venv venv
source venv/bin/activate  # Windows: venv\Scripts\activate
pip install -r requirements.txt

# 2. Configure API keys
cp .env.example .env
# Edit .env and add your API keys

# 3. Start backend
uvicorn app.main:app --reload

# 4. Install VS Code extension
cd ../vscode-extension
npm install
npm run compile

# 5. Press F5 in VS Code to start
```
