## üñ®Ô∏è Printing Instructions

### For Presentations:
1. Open **iTechSmart_Architecture_Simple.html**
2. Press **Ctrl+P** (Windows) or **Cmd+P** (Mac)
3. Select **"Save as PDF"**
4. Choose **Landscape** orientation
5. Set margins to **None**
6. Enable **Background graphics**
7. Save as **iTechSmart_Architecture.pdf**

### For Posters:
1. Use the same process
2. Select **A3** or **Tabloid** paper size
3. Print at **300 DPI** for best quality
