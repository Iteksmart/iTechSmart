**Classification Criteria:**
```python
def classify_incident(event):
    if event.type == 'data_exfiltration':
        return 'CRITICAL'
    elif event.type == 'unauthorized_access' and event.success:
        return 'HIGH'
    elif event.type == 'failed_login' and event.count > 10:
        return 'MEDIUM'
    else:
        return 'LOW'
```
