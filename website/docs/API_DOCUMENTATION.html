iTechSmart Suite - API Documentation

Overview

The iTechSmart Suite provides a comprehensive REST API for programmatic access to all platform features.

Authentication

All API requests require authentication using JWT tokens.

Authorization: Bearer YOUR_JWT_TOKEN

Obtaining a Token

POST /api/v1/auth/token 
{
  "username": "your_username",
  "password": "your_password"
}

Core Endpoints

Monitoring

GET /api/v1/monitoring/metrics

Retrieve system metrics and performance data

GET /api/v1/monitoring/alerts

Get active alerts and notifications

Incident Management

GET /api/v1/incidents

List all incidents

POST /api/v1/incidents

Create a new incident

{
  "title": "Database connection timeout",
  "severity": "high",
  "description": "Production database experiencing connection timeouts",
  "assigned_to": "user_id"
}
PUT /api/v1/incidents/{id}

Update an existing incident

Automation

GET /api/v1/automation/workflows

List all automation workflows

POST /api/v1/automation/workflows

Create a new workflow

{
  "name": "Auto-scale on high CPU",
  "trigger": "cpu_threshold",
  "actions": [
    {
      "type": "scale_up",
      "parameters": {"instances": 2}
    }
  ]
}

Cloud Management

GET /api/v1/cloud/resources

List cloud resources across all providers

POST /api/v1/cloud/deploy

Deploy resources to cloud providers

Response Format

All API responses follow this standard format:

{
  "status": "success",
  "data": {
    // Response data
  },
  "message": "Operation completed successfully",
  "timestamp": "2024-11-17T10:15:00Z"
}

Error Handling

Error responses include detailed information:

{
  "status": "error",
  "error": {
    "code": "INVALID_REQUEST",
    "message": "Missing required field: title",
    "details": {}
  },
  "timestamp": "2024-11-17T10:15:00Z"
}

Rate Limiting

API requests are rate-limited to:

  • 1000 requests per hour for authenticated users
  • 100 requests per hour for unauthenticated requests

Webhooks

Configure webhooks to receive real-time notifications:

POST /api/v1/webhooks 
{
  "url": "https://your-app.com/webhook",
  "events": ["incident.created", "alert.triggered"],
  "secret": "your_webhook_secret"
}

SDK Support

Official SDKs available for:

  • Python: pip install itechsmart-sdk
  • JavaScript/Node.js: npm install @itechsmart/sdk
  • Go: go get github.com/itechsmart/sdk-go
