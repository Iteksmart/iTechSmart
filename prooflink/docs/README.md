# ProofLink.AI

## Overview

Document Verification - AI-powered document authentication

## Features

- **Document verification**
- **Blockchain timestamping**
- **Digital signatures**
- **Tamper detection**
- **Version control**
- **Audit trails**
- **Compliance reporting**
- **Multi-format support**


## Technology Stack

FastAPI, PostgreSQL, Blockchain, AI/ML

## Installation

### Using Docker (Recommended)

```bash
cd prooflink
docker-compose up -d
```

### Manual Installation

**Backend:**
```bash
cd backend
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt
python main.py
```

**Frontend:**
```bash
cd frontend
npm install
npm run dev
```
