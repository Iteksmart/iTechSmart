# ImpactOS

## Overview

Impact Measurement - Social and environmental impact tracking

## Features

- **Impact metrics tracking**
- **SDG alignment**
- **ESG reporting**
- **Stakeholder management**
- **Impact visualization**
- **Custom frameworks**
- **Automated reporting**
- **Benchmark comparison**


## Technology Stack

FastAPI, PostgreSQL, Analytics engine

## Installation

### Using Docker (Recommended)

```bash
cd itechsmart-impactos
docker-compose up -d
```

### Manual Installation

**Backend:**
```bash
cd backend
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt
python main.py
```

**Frontend:**
```bash
cd frontend
npm install
npm run dev
```

## Access Points
