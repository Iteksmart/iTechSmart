## 1. Getting Started

### Creating Your Account

1. **Visit the Registration Page**
   - Go to https://app.impactos.com/auth/register
   - Or click "Get Started" on the homepage

2. **Fill in Your Information**
   - Email address (will be used for login)
   - Username (unique identifier)
   - Full name
   - Password (minimum 8 characters, must include uppercase, lowercase, number, and special character)

3. **Verify Your Email**
   - Check your email inbox for verification link
   - Click the link to activate your account
   - You'll be redirected to the login page

4. **First Login**
   - Enter your email and password
   - Click "Sign In"
   - You'll be taken to the onboarding wizard

### Initial Setup Wizard

After your first login, you'll be guided through:

1. **Organization Setup**
   - Enter your organization name
   - Add organization details (mission, website, etc.)
   - Upload your logo (optional)

2. **Team Invitation**
   - Invite team members by email
   - Assign roles (Admin, Program Manager, Grant Writer, etc.)

3. **First Program**
   - Create your first program
   - Set goals and metrics
   - Define target population

## 4. Program Management

### Creating a Program

1. Navigate to "Programs" section
2. Click "+ New Program"
3. Enter program details:
   - Program name
   - Category (Education, Health, Environment, etc.)
   - Description
   - Target population
   - Geographic area
4. Set goals and objectives
5. Define metrics and KPIs
6. Set budget and timeline
7. Click "Create Program"

### Managing Program Metrics

**Adding Metrics**
1. Open program details
2. Go to "Metrics" tab
3. Click "+ Add Metric"
4. Enter:
   - Metric name
   - Description
   - Unit of measurement
   - Target value
   - Collection frequency
5. Click "Save Metric"

**Updating Metric Values**
1. Find the metric in the list
2. Click "Update Value"
3. Enter current value
4. Add notes (optional)
5. Click "Save"

**Viewing Progress**
- Progress bars show achievement rate
- Charts display trends over time
- Color coding indicates status (green = on track, yellow = at risk, red = behind)

## 5. Impact Reporting

### Generating an Impact Report

1. Navigate to "Reports" section
2. Click "+ Generate Report"
3. Select report type:
   - **Quarterly Report**: 3-month summary
   - **Annual Report**: Year-end comprehensive report
   - **Donor Report**: Customized for donors
   - **Grant Report**: For grant funders
4. Choose program
5. Select reporting period
6. Click "Generate with AI"
7. Wait for AI to create report (1-2 minutes)

### Customizing Reports

**Editing Content**
1. Open generated report
2. Click "Edit" on any section
3. Modify text as needed
4. Click "Save Changes"

**Adding Sections**
1. Click "+ Add Section"
2. Choose section type
3. Enter content or use AI assistant
4. Click "Add to Report"

**Adding Visuals**
1. Go to "Visualizations" tab
2. Select chart type
3. Choose data to display
4. Customize colors and labels
5. Click "Add to Report"

### Exporting Reports

**PDF Export**
1. Open report
2. Click "Export" button
3. Select "PDF"
4. Choose layout (portrait/landscape)
5. Click "Download PDF"

**Sharing Reports**
1. Click "Share" button
2. Choose sharing method:
   - Email (enter recipient emails)
   - Public link (generate shareable URL)
   - Download (save to computer)
3. Set permissions (view only/edit)
4. Click "Share"

## 6. Grant Management

### Finding Grant Opportunities

1. Navigate to "Grants" section
2. Click "Search Grants"
3. Enter search criteria:
   - Focus area
   - Amount range
   - Deadline
   - Funder type
4. Click "Search"
5. Browse results
6. Click "Save" to bookmark interesting grants

### Creating a Grant Proposal

**Using AI Assistant**
1. Open saved grant
2. Click "Create Proposal"
3. Select organization and program
4. Click "Generate with AI"
5. AI will create:
   - Executive summary
   - Problem statement
   - Methodology
   - Budget
   - Evaluation plan
6. Review and edit as needed

**Manual Creation**
1. Click "+ New Proposal"
2. Select grant opportunity
3. Fill in each section manually
4. Save draft frequently
5. Submit when complete

### Tracking Proposals

**Proposal Dashboard**
- View all proposals
- Filter by status (Draft, Submitted, Under Review, Approved, Rejected)
- Sort by deadline
- Track submission dates

**Deadline Reminders**
- Automatic email reminders
- Dashboard notifications
- Calendar integration

## 10. Troubleshooting

### Common Issues

**Can't Log In**
- Verify email and password
- Check caps lock
- Try "Forgot Password"
- Clear browser cache
- Try different browser

**Report Generation Fails**
- Check internet connection
- Verify program has data
- Try different date range
- Contact support if persists

**Slow Performance**
- Clear browser cache
- Close unused tabs
- Check internet speed
- Try different browser

**Missing Data**
- Refresh page
- Check filters
- Verify date range
- Ensure proper permissions
